# Keep Code

Effortlessly store, manage, and retrieve your code snippets your web browser. [https://keep-code.vercel.app/](https://keep-code.vercel.app/)

## Install

```bash
git clone https://github.com/brandonporcel/keep-code.git
cd keep-code
npm install
npm run dev
```
